# utils
utils for programming contest

## use template
 - run script (and fetch test case (see atcoder))
 ```
./use_template.sh {filename} {URL}
 ```

## test case
 - create test directory
 - store each test case to test\_/{a, b, c..}
 - run script
 ```
  ./test_case.sh {a, b, c...}
 ```
